# conwey_life_game
此项目实现了一个康威生命游戏的基本运行以及一些模式，建议使用Visual Studio 2022进行运行，如有bug请自行改动，该项目可能不会更新./This project implements the basic operation of Conway's Game of Life along with some patterns. It is recommended to run it using Visual Studio 2022. Please modify it yourself if any bugs are found. Please note that this project might not receive further updates.
